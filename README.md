## 🚀 Featured Projects

### 1. **FinTrustRAG** — Hallucination-Aware Financial RAG System *(Apr 2026)*
- Hybrid RAG with FAISS + BM25 + Reciprocal Rank Fusion
- Custom **Numerical Consistency Trust Scorer (NCTS)** | T = 0.6×G + 0.4×M
- **84% hallucination detection rate** on FinanceBench vs. 43% for naive RAG
- Stack: FastAPI · React/Vite · Groq/Llama-3.1-8B · FAISS
- [Repository](https://github.com/NikithaDineshKumar/FinTrustRAG)

### 2. **Personal Finance Tracker** — Full-Stack MERN + Gemini AI *(Jun 2026)*
- JWT authentication, expense/budget management, visual dashboards
- AI features: auto-categorization, Smart Spending Insights, RAG-based Budget Advisor Chatbot
- PDF export via jsPDF
- **Deployed**: [Frontend](https://finance-tracker-gray-xi.vercel.app) (Vercel) | Backend (Render)
- Stack: MongoDB · Express.js · React · Node.js · Gemini API · Recharts

### 3. **pdfToPng** — Open Source Contribution (GSSoC 2026)
- Identified and fixed a bug where the working PDF-to-Word feature was missing from the app's tool registry, making it invisible in the UI
- [PR #380 merged](https://github.com/Durgeshwar-AI/pdfToPng/pull/380) | [GSSoC Profile](https://gssoc.girlscript.tech/)

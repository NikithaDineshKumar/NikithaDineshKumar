## 🚀 Featured Projects

### 1. **FinTrustRAG** - Hallucination-Aware Financial RAG System
- Hybrid RAG with FAISS + BM25 + Reciprocal Rank Fusion
- Custom **Numerical Consistency Trust Scorer (NCTS)** | T = 0.6×G + 0.4×M
- **84% hallucination detection rate** on FinanceBench
- Stack: FastAPI, React/Vite, Groq/Llama-3.1, FAISS
- [Repository](https://github.com/NikithaDineshKumar/FinTrustRAG)

### 2. **Personal Finance Tracker** - MERN + Gemini AI
- JWT authentication, expense CRUD, budget management
- AI-powered expense categorizer & Smart Spending Insights
- RAG Budget Advisor Chatbot
- PDF export via jsPDF
- **Deployed**: [Frontend](https://github.com/NikithaDineshKumar/) (Vercel) | [Backend](https://github.com/NikithaDineshKumar/) (Render)
- Stack: MongoDB, Express, React, Node.js, Gemini API, Recharts
